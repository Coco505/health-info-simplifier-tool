## Overview

HealthInfoSimplifier helps healthcare professionals, medical writers, and patient educators convert complex medical information into accessible content. Powered by Gemma 3, it analyzes readability, simplifies text to appropriate grade levels, and translates into 27+ languages.

## How to Use

1. **Input** - Paste text or load sample content
2. **Simplify** - Click "Simplify Text" to generate patient-friendly version
3. **Refine** - Use "Simpler" or "Detailed" buttons to adjust complexity
4. **Translate** - Select target language and click "Go" (optional)
5. **Copy** - Copy results to clipboard
